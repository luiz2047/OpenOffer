# OpenOffer Documentation

OpenOffer is a local-first, free, open-source assistant for interviews, career
work, meetings, and related desktop workflows.

## Legacy archive

The `docs/engineering/` folder preserves pre-rebrand engineering notes,
rollout records, and migration writeups from the Natively-era codebase. Those
documents are kept for provenance and historical reference, not as current
product guidance.

Use the current product docs first. Treat archive files as background material.
