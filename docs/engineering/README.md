# Engineering Archive

This folder is a legacy archive of engineering notes from the pre-rebrand
Natively-era project. OpenOffer keeps these documents for provenance and
historical context only.

The archive includes migration notes, parity reports, relay writeups, and
other implementation records that describe hosted-control-plane, relay, or
commercial-era behavior that is no longer part of the public OpenOffer
identity.

Read the top-level documentation first. Use these files as background material
when you need historical detail, attribution, or migration context.
